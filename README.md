# Blackbox Testing

Implementation of Black Box Testing methods for a Room Booking System.

## Topics
This project implements several Black Box Testing techniques:

- Equivalence Partitioning (EP)
- Boundary Value Analysis (BVA)
- Decision Table Testing
- State Transition Testing

---

## Study Case
Room Booking System for university discussion rooms.

### Requirements
- Booking duration must be between 1–4 hours
- Students can book rooms only if:
  - Registered
  - No violations
  - Room available
- Room transitions must follow valid states
